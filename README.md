# LINE 禮物優惠券庫存監測小工具

這是一個用於即時查詢 LINE 禮物優惠券剩餘數量的瀏覽器書籤小工具 (Bookmarklet)。

## 功能特色
* **視覺化介面**：直接在網頁右上方彈出資訊視窗，不需開啟開發者工具。
* **即時庫存**：攔截底層 API，精準顯示每款優惠券的總發行量、已領取數與剩餘數量。
* **狀態提示**：包含視覺化進度條，以及「已搶空」、「已領取」、「暫不可領」等狀態標示。
* **自動更新**：內建 30 秒自動倒數更新機制，並支援手動點擊刷新。

## 安裝教學
1. 在瀏覽器（如 Chrome, Edge, Safari）的書籤列上按右鍵，選擇「新增網頁」或「新增書籤」。
2. 名稱可以自訂，例如輸入「🎁 LINE 禮物庫存」。
3. 在「網址 (URL)」欄位中，貼上本專案 `bookmarklet.js` 檔案裡面的**所有程式碼**，然後點擊儲存。

## 使用方法
1. 使用電腦版瀏覽器，前往 LINE 禮物的「可領取」優惠券頁面。
2. 點擊你剛剛建立好的「🎁 LINE 禮物庫存」書籤。
3. 畫面右上方就會自動跳出監測視窗，開始抓取並顯示目前的票券剩餘狀態！
